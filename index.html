Connect Google Calendar
1
Create a Google Cloud project
Go to console.cloud.google.com → New Project → name it STH Planner
2
Enable the Google Calendar API
APIs & Services → Enable APIs → search Google Calendar API → Enable
3
Create an OAuth Client ID
APIs & Services → Credentials → Create Credentials → OAuth Client ID → Web Application

Under Authorised JavaScript origins add the URL where this app is hosted (e.g. https://yourusername.github.io)

Copy the Client ID and paste it below:
4
Configure OAuth consent screen
APIs & Services → OAuth consent screen → External → fill in app name → Add your Google account as a Test user → Save
